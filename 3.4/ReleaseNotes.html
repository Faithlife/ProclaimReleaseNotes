3.4.0.0003 Tuesday, June 7, 2022

New to Proclaim

  • Support multi-video playback. Use the Video option on an alternate content virtual screen to play multiple videos at once, or use Alternate to show a motion background along with a video item.
  • Add media export from the Media Browser. Right-click the media you want to download, then click "Export Media"
  • Support 60hz support for SDI and NDI, as well as a framerate selection option to NDI settings.
  • Add new fonts: Aguafina Script, Aladin, Amarante, Bahiana, Bangers, Beau Rivage, Bokor, Cormorant Garamond, DM Serif Text, Federo, Frederick the Great, Great Vibes, IM Fell English SC, Italiana, Metal Mania, Mouse Memoirs, Pirata One, Press Start 2P, Reenie Beanie, Romanesco, Trade Winds, Unical Antiqua, ZCOOL Kuaile

Improvements and Bug Fixes

  • Fix sync issue that caused song items to revert to an older version.
  • Prevent crash when attempting to access a corrupt database.
  • Fix memory leak and subsequent crash with NDI on Windows 11.
  • Various performance improvements when opening a presentation.
  • Respect selected confidence timer settings.
  • Show option to remove foreground on alternate content with multi-selected songs.
  • Show option to restore original on alternate content when multi-selected.
  • Support multiple canvas sizes for margins and styles on media templates.
  • Don't regenerate the slide when its backing media finishes uploading.
  • Fix unclickable checkboxes on older macOS versions.
  • Fix warping foreground images when creating Smart Media from a slide.
  • Respect differences in text box styles between formats in Smart Media Editor.
  • Add 'Restore Text' option to alt content.
  • Fix motion background coming into Smart Media Editor as a still image.
  • Make sure the Warm Up starts on time even if a different slide change occurs at the same time.
  • Ensure correct text field styles/margins are used from smart media on alt content.
  • Prevent odd behavior when typing in an Announcement V2 (Mac only).
  • Fix unselectable description field on Group Upload media (Mac only).
  • Prevent crash while removing a virtual screen while multi-selected.
  • Fix infrequent crash calculating times for nested groupings.
