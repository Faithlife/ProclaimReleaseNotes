Proclaim Version 4.1

4.1.0.0050 Monday, February 5, 2024

  • Prevent crash adding and removing service items in Greenwich Mean Time.
  • Fix crash rendering text in non-latin fonts.
  • Mitigate incorrect first-frame issue with videos on Windows.

4.1.0.0012 Friday, January 19, 2024

  • Remove inapplicable settings from alt-content Edit view.
  • Make sure Add images button always visible during batch upload (Mac only).

4.1.0.0008 Wednesday, January 17, 2024

New Features

  • Allow transitions styles and times to differ per virtual screen.
  • Allow changing time on lyric scrolling animation.
  • Simplify options for adding media to a presentation.
  • Add support for the next verse button to On Screen Bible items.
  • Enable plus and minus keys to advance and retreat the slides.

Bug Fixes

  • Improve experience with Planning Center lyric import and worship tools.
  • Improve connection between Proclaim and a linked SongSelect account and add logging.
  • Be smarter with slide index when changing Bible passage in preview mode.
  • Fix crash loading Lighting Scene (Windows only).
  • Prevent crash toggling service item indices without an open presentation.
  • Fix crash importing media when the import window is already open (Mac only).
  • Prevent crash going On Air to a display that's no longer connected.
  • Prevent crash trying to load a timer that doesn't exist.
  • Prevent crash choosing foreground image.
  • Prevent crash opening the confidence editor when the window is already open.
  • Prevent rare crash joining a presentation with the Remote.
  • Make sure the Confidence view always shows the background image when slide content is blank.
  • Show tooltips when button is disabled (Windows only).
  • Show the correct color number on presentation lock icon (Windows only).
  • Truncate older presentation dates in the Open menu (Windows only).
  • Restore transition info when clicking Restore Original on an alternate content virtual screen.
  • Be clearer about which presentation media items aren't licensed.
  • Only show unlicensed media locks on their respective virtual screens.
  • Fix error submitting an updated sermon recording.
  • Remove duplicate Add item buttons in Suggested Media.
  • Fix issue preventing copy-paste from Proclaim into some other programs (Mac only).
  • Clarify confidence monitor settings when they cannot be modified.

Remote Improvements

  • Stop sending hidden service items to local wifi Remote connections.
  • Show black background in confidence mode over a local-wifi connection.
