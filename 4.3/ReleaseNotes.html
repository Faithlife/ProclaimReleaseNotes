Proclaim Version 4.3

4.3.0.0221 Wednesday, May 1, 2024

  • Fix crash opening the countdown clock display settings (Windows only).

4.3.0.0220 Tuesday, April 30, 2024

  • Fix crash opening the About menu with Spanish as the selected language.
  • Prevent clipping on updated Spanish strings.
  • Preserve foreground image z-index across aspect ratios.
  • Prevent crash when sharing a service item or setlist multiple times.

4.3.0.0210 Monday, April 29, 2024

New

  • Send foreground images and text boxes forward and backward on the canvas.
  • Find these settings in the Arrange menu, or right- or control-click a foreground element to view them.

Improved

  • Show in-progress uploads in the media browser.
  • Allow Lyric Scrolling as the default transition for songs.
  • Make on-screen messages optional on alternate content virtual screens.
  • Allow sign feed deletion from inside the application.
  • Change name of cloned items to linked items and show a banner when selected.
  • Improve foreground element selection. Click on each overlapping section to select each item from front-to-back.

Bug Fixes

  • Prevent sermon recordings from accidentally being linked to someone you don't recognize.
  • Allow a display to be switched between Slides and Notes outputs while On Air.
  • Prevent SongSelect from automatically signing you in when re-linking or changing accounts.
  • Allow Recently Used media to be deleted.
  • Prevent incorrect song linking while importing lyrics.
  • Make sure the right linked-item options are shown.
  • Fix rare crash parsing invalid confidence timer time.
  • Fix rare crash opening app.
