3.2.0.0002 Monday, September 27, 2021

  • Rework alt-content display options for more flexibility.
    • Allow Main Slide content to be different on alternate screens. Now, you can change four lines of lyrics to two by removing slide breaks, show your content in another language, or add a different bible translation.
    • Simplify display options to Slide and Blank.
    • Change Background only and foreground only from options in a menu to clickable text.
    • Add an option to return alt content to original content.
    • Make OSB, Videos and other items fully customizable in alt content.
  • Support Key / Fill for internal and external keying on supported Black Magic devices.
  • Include built-in support for Stream Deck with our companion Stream Deck Plug-in. Learn more
  • Add ability to clone service items, not just duplicate them.
    • Add clone option to each service item when right-clicked in the Order of Service. Clones of an item operate as if they are all the same, and changes made to a cloned item appear on all of its clones.
    • Add quick options to clone items to pre- and post-service loops.
  • Organize sections and groupings into collapsible sections in grid preview.
  • Improve performance when loading and viewing a presentation.
    • Load a selected item first to improve first time slide-generation experience, like viewing a presentation for the first time on a computer.
    • Make better use of available system resources for improved slide generation speed.
    • Mac Only: Grid Preview now has responsive scrolling. Adopts Apple-provided API to provide a smooth scrolling experience and fixes terrible lag on 4K and 5k displays such as the 5k iMac.
  • Add the option to apply a background to multiple selected items regardless of their type. Select the whole presentation and apply a consistent background in one click.
  • Render foreground images immediately when dragged in or pasted from the clipboard.
  • Fix bug that causes us not to show example text in Default Media and Default Confidence settings.
  • Fix bug that causes motion backgrounds on alt-content screens to not respect the chosen media layout setting.
  • Retain the selected tab across service item changes.
  • Fix bug that causes Powerpoint images to be imported at a lower resolution.
