Proclaim Version 3.5

3.5.0.0027 Monday, January 24, 2023

  • Fix Mac crash when launching from local user account.
  • Fix rare crash on Windows when launching from computers with long-standing installations of Proclaim.
  • Fix crash resizing the Media Browser so small that it doesn't show any media.

3.5.0.0018 Monday, January 23, 2023

  • Add Native Apple Silicon support.
  • Improve scrolling performance of Media Browser on high-DPI screens.
  • Allow default media for Video Items.
  • Reopen song results window on click in the text box.
  • Add tooltip to clarify \`No Arrangement\` Option.
  • Fix rare crash when rendering text with certain fonts.
  • Don’t require scrolling to see all options when Timeline is expanded (Windows-only).
  • Prevent crash when going on air with an empty or invalid Live Stream Cue (Windows-only).
  • Select reference field after creating a Bible item to allow instant reference entry.
  • Fix crash undoing a just-added video to a Video item.
  • Improve how arrangement names are created.
  • Remember last selected visibility state for title and credits.
  • Remove 'future' AppCommand listed in UI.
  • Improve performance when switching presentations.
  • Fix bug preventing Slideshow Background settings from staying open (Mac Ventura).
  • Restore ability to upload supported video formats from unsupported platforms.
  • Fix crash handling 'completed' transcode job in bad state.
  • Fix Invite button that overlaps other controls.
  • Make sure unsynced arrangements are handled correctly when Proclaim opens.
  • Ensure that old arrangements don’t blow away changes to new ones.
