Proclaim Version 3.0

3.0.0.0023 Monday, August 24, 2020

  • New smart media editor for announcement items.
  • Allow deleting Faithlife Music Arrangements.
  • Improve mixed-multi select UI on Mac.
  • Assorted bug fixes.

3.0.0.0022 Thursday, August 20, 2020

  • Fix issue that could cause remote to get out of sync with Proclaim.
  • Fix issues with dpi causing NDI output on Windows to be blank.
  • Fix rare crash with song dropdown.

3.0.0.0017 Friday, August 14, 2020

  • Restore saving NDI as custom output on Mac.
  • Close virtual windows when exiting Proclaim on Windows.

3.0.0.0015 Tuesday, August 11, 2020

  • Restore countdown clock on Mac.
  • Fix potential crash for older versions of Windows when using NDI.

3.0.0.0011 Friday, August 7, 2020 (Windows Only)

  • Fix issue preventing mouse clicks on web page service items.
  • Prevent crash when attempting to use NDI on an unsupported version of Windows.

3.0.0.0009 Thursday, August 6, 2020

  • Fix crash when video input device is missing. (Windows)
  • Fix crash when clicking VirtualScreen reorder control. (Mac)
  • Prevent spacebar on Windows from jumping back to a seemingly random slide.
  • Fix timing issue that could cause a crash with the video countdown timer.

3.0.0.0005 Wednesday, August 5, 2020 (Windows Only)

  • Fix start menu path for Proclaim on Windows.

3.0.0.0004 Wednesday, August 5, 2020

  • Add new Virtual Screen display settings.
  • Improve performance and stability of video input on Windows.
  • Allow alternate content for foreground only, background only, blank, and duplicate on a per item basis.
  • Support NDI for slides and alt content outputs.
  • Allow different aspect ratios for different outputs (e.g. 16x9 slides and 4x3 confidence).
  • Add option to show videos on the Confidence monitor.
  • Add video countdown confidence item for Confidence templates.
  • Show a dark checkerboard for transparency when the text content is light.
  • Support drawing on slides with the Proclaim Remote (pre-released in 2.12).
  • Refresh icons to be more consistent with the rest of Faithlife Integrated Ministry Platform.
    • The following requires your group to have its songs migrated to Faithlife Music. The migration process is automatic. If you don't notice these new features, email us at support@faithlfe.com and we can check on the status of your group.
  • Integrate with Faithlife Music, where you can edit and manage your songs and arrangements on the web at music.faithlife.com. Any changes sync between Proclaim and Faithlife Music.
  • Add multiple arrangements to a single song (different verse order and lyrics).
  • Show in-line chords with ChordPro syntax to lyrics to show on the Confidence monitor.
  • Allow separate display title, for use on slides, separate from the actual title of the song. (e.g. 'Before the Throne' on slide, but song title 'Before the Throne of God Above').
