Proclaim Version 4.2

4.2.0.0206 Tuesday, March 5, 2024

New Features

Improved In-App Support for Digital Signage
  • Support sign feed creation without going to Faithlife.com. Learn more
  • Show current sign feed in Proclaim's built-in web browser, including current verse art selections. Videos, Bible Trivia, and Web Page items are not supported in the Preview, but can be viewed in a browser like Chrome, Safari, or Firefox.
  • Prevent blank sign in feed while editing.
  • Exit service on sign feeds after Proclaim closed without going Off Air.
  • Allow access to group uploads from your church's other presentation teams. You must be a member of other teams to see their group uploads.

Improved

Confidence Timer
  • Show confidence timer info the Order of Service.
  • Use 5 min as the new default value for confidence timer cue
  • Make sure confidence timers update for Daylight Savings time.
  • Show processing spinner in font menu for active uploads.
  • Improve offline media searching by limiting offline browsing to recent uploads.

Bug Fixes

  • Make sure Proclaim saves recordings when it is closed without going Off Air.
  • Correctly scale foreground images when added from the media browser.
  • Show countdown clock, messages, and overlays on mirrored alt-content virtual screens (Mac only).
  • Fix countdown rendering bug (Mac only).
  • Clean up text box spacing in Smart Media and Countdown editors (Windows only).
  • Preserve active countdown timers through embedded Bible slides.
  • Fix crash opening countdown editor in a presentation using an aspect ratio other than 16:9 or 4:3.
  • Show selected text colo(u)r in colo(u)r picker box.
  • Make sure tab selection works consistently in Edit mode (Windows only).
  • Save hymn number to arrangement.
  • Prevent the Proclaim Open Presentation screen from flashing 'get started' while loading.
  • Fix subscription info link.
  • Display any active trial licenses during checkout (Mac only).
