Proclaim Version 2.4

2.4.0.0014 Tuesday, September 5, 2017

  • Lighting Control - Add lighting scenes to your presentation to automatically control lighting systems using MIDI and Philips Hue configurations.
  • Inbound MIDI - Control your presentation by configuring Proclaim to listen to inbound MIDI signals.
  • Add support for adding Signals to cues.
  • Fix recording scope issue that could cause the wrong section to be recorded.

2.4.0.0010 Tuesday, August 29, 2017

  • Prevent recording pause when showing Quick Screen

2.4.0.0008 Wednesday, August 16, 2017

  • Fixed bug with adding text box and foreground images
  • Keep song's title and main slide example in sync
  • Render input text on song items
  • Added warnings added to Windows 'Save Template' dialogue.

2.4.0.0004 Monday, August 14, 2017

  • Fixed crash at start-up

2.4.0.0003 Monday, August 14, 2017

  • Smart Media Creator - create your own Smart Media templates for use by your group.
  • Smart Media Defaults - use the Settings dialog to set a default Smart Media template for each Order of Service item type.
  • Removed automatic Smart Media defaulting for newly created Order of Service items. All Smart Media will default to black unless manually overridden (see previous bullet).
  • Added "Fade In" and "Fill in the blank" toolbar buttons for Rich Text editing.
  • Enabled Auto Advance, Custom Transition and Countdown Clock for Bible Trivia.
  • Changed default presentation aspect ratio to 16:9.
  • Removed suggested media from Background popup menu.
