Proclaim Version 4.5

4.5.0.0186 Monday, August 12, 2024

New

Increased Support for Chords in the Confidence Monitor
  • Add option to import ChordPro chords from SongSelect with a SongSelect Advanced or Premium subscription.
  • Support in-app chord transposition via the key selector.
  • Add major and minor key distinction to lyric import and arrangement key dropdowns.
  • Show lyric and ChordPro previews to SongSelect import window.
  • Remove hyphenated lyrics from slides if chords are present.

Improved

  • Restyle watermark for unlicensed content and On Air sessions.
  • Standardize inline square shape across slides.

Bug Fixes

  • Sync arrangement key from Proclaim to music.proclaim.logos.com.
  • Ensure we import all sections of a song from SongSelect.
  • Clear SongSelect credentials after failed login.
  • Fix error sending invitations from Team menu.
  • Keep current slide on screen when zooming in Grid Preview.
  • Fix failed PowerPoint presentation import when the computer language is different that Proclaims.
  • Make sure Confidence Template isn't empty by default.
  • Prevent unintentional aspect ratio changes to new presentations.
  • Fix crash migrating virtual screen settings.
  • Fix clipping on sort option in the Media Browser (Windows only).
  • Fix crash changing On Screen Bible version to a Bible with an unsupported reference.
  • Make sure the first service item is selected when switching to a presentation that starts with a grouping.
  • Scroll Order of Service up when active item is off screen (Mac only).
  • Restore toggle indicator for Show on All Slides in image context menu (Windows only).
